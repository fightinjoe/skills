# Agent Skills

A collection of agent skills that extend capabilities across planning, development, and tooling.

## App creation

- **app-init** - Initialize a new app, creating a brief and outlining open questions and prototypes to help define them
- **app-architect** - Takes an app brief and creates an architectural plan for structuring the app
- **app-review-brief** - Revisit and update a BRIEF.md after prototyping to reflect what was actually learned
